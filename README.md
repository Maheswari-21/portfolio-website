# Portfolio Website

A modern responsive portfolio website built to showcase my skills, projects, experience, and resume as a Full Stack MERN Developer.

## About The Project

This portfolio website is developed using Next.js and Tailwind CSS to present my professional profile, technical skills, projects, and contact information.

The main goal of this project is to create a personal brand website where recruiters can easily explore my work and development experience.

## Tech Stack

- Next.js
- React.js
- JavaScript
- Tailwind CSS
- Git & GitHub

## Features

- Responsive design for mobile and desktop
- Hero section with introduction
- About section
- Technical skills showcase
- Project showcase with details
- Resume download option
- Contact section
- Modern UI design

## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Maheswari-21/portfolio-website.git

```
Navigate to project directory:

```bash
cd portfolio-website
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
